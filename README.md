# Smriti Giri | Portfolio

Personal portfolio website showcasing my projects, skills, certifications, and experience as a Python/Django and full-stack web developer.



## Technologies Used
- HTML5
- CSS3
- JavaScript (vanilla)
- Tailwind CSS
- FormSubmit (contact form email delivery)

## Features
- Responsive design (mobile, tablet, desktop)
- Dark / light mode toggle with saved preference (localStorage)
- Animated hero section with particle background (HTML5 Canvas)
- Typing animation for role titles
- Scroll-triggered fade-in animations (IntersectionObserver)
- Active-section highlighting in navigation while scrolling
- Project showcase with tech-stack tags and GitHub/live links
- Certifications & trainings section
- Community & involvement section
- Downloadable CV/resume
- SEO optimized: meta tags, Open Graph, Twitter cards, JSON-LD structured data.

## Projects Featured
1. **Annapurna Circuit Tourism Website** — Multi-page tourism site (HTML, CSS, responsive design)
2. **Mitho Koseli — E-Commerce Platform** — Full e-commerce app with cart, payments, admin panel, normalized MySQL schema (PHP, MySQL, JavaScript)
3. **To-Do List Web Application** — Django task manager with CRUD, ORM, SQLite, per-user auth
4. **Weather Mobile Application** — Android app consuming the OpenWeather API (Java, Android Studio, REST API)
5. **Job Tracker Web Application** — Django app tracking job applications with a Bootstrap dashboard
6. **Task Management System** — Django app with role-based workflows (Admin / Supervisor / Intern)
7. **Student Registration System** — Registration system storing submissions to Google Sheets via Apps Script, hosted on Cloudflare (Live)
