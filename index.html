About Me

Hi, I'm Smriti — a Bachelor of Information Technology student at Gandaki University, Pokhara, with a genuine love for turning ideas into working software. My focus is Python and Django, but I'm comfortable moving across the stack, from responsive front-ends to relational database design.

Over the past few semesters I've built and shipped seven projects — tourism sites, e-commerce platforms, task and job trackers, and mobile apps — while picking up REST APIs, machine learning basics, and Flutter along the way. I care about clean, readable code and interfaces that feel effortless to use.

Outside the classroom, I'm an active member of Code for Change Pokhara and the BEE_IT HIVE club, where I help organise hackathons and workshops. I'm currently seeking an internship where I can keep learning while contributing to real products.
